# To-Do-List-App
Simple To-Do List App 📝
A lightweight and intuitive web application designed to manage daily tasks efficiently. This project focuses on providing a seamless user experience through a clean interface and real-time task management.
🚀 Features
• Add Tasks: Quickly add new items to your list.
• Remove Tasks: Delete completed or unwanted tasks with a single click.
• Instant Updates: The list updates dynamically without refreshing the page.
• Responsive Design: Fully optimized for both desktop and mobile screens.
🛠 Tech Stack
• HTML5: Structure and semantic content.
• CSS3: Modern styling with a focus on clean UI/UX.
• JavaScript (Vanilla JS): Core logic for DOM manipulation and task handling.
📂 How to Run
Since this is a client-side application, no installation or local server is required:
1.	Download or clone the repository.
2.	Open the index.html file in any modern web browser.
💡 Key Concepts Applied
• DOM Manipulation: Dynamically creating and removing elements using JavaScript.
• Event Handling: Managing user interactions such as button clicks and keyboard inputs.
• Responsive Layouts: Using CSS techniques to ensure the app looks great on all devices.
